MNKnowledgeBase - 搜索可视化框架

说明

这个目录包含一个用于可视化搜索界面的静态原型：`search.html`。
它用于在不接入真实后端或索引文件的情况下验证交互和界面布局。

如何打开

1. 在本地直接用浏览器打开 `ui/search.html`：
   - Finder 中双击打开，或在终端中运行：
     open ui/search.html

2. 建议使用 Chrome / Safari 打开以获得更好的体验（需要剪贴板权限以支持复制）。

功能（当前静态实现）

- 顶部固定搜索框：页面滚动时保持可见。
- 预设词条（pill）：点击高亮表示选择，选中的预设会同步到搜索输入并参与搜索。
- 搜索结果：展示卡片的标题、类型与颜色条，并支持单选高亮。
- 工具栏：对选中卡片执行定位、复制 URL、复制 Markdown 链接及自定义操作（当前以 alert/剪贴板模拟）。

后续接入建议

- 将 `fakeCards` 替换为从现有索引文件（JSON）或插件接口异步获取的数据。
- 把 `doSearch` 替换成调用实际的搜索器（例如 `KnowledgeBaseSearcher`）并把返回结果渲染在页面上。

- 使用 WebSocket / postMessage 与 MarginNote 插件通信以实现直接定位卡片与操作。
